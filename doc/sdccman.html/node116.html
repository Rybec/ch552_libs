Besides defines from the C standards, the compiler creates the following #defines:

#define
Description
__SDCC
Always defined. Version number string (e.g. SDCC_3_2_0 for sdcc 3.2.0).
SDCC

__SDCC_mcs51 or __SDCC_ds390 or __SDCC_z80, etc.
depending on the model used (e.g.: -mds390). Older versions used SDCC_mcs51, etc instead.
__SDCC_STACK_AUTO
when —stack-auto option is used
__SDCC_MODEL_SMALL
when —model-small is used
__SDCC_MODEL_MEDIUM

__SDCC_MODEL_LARGE

__SDCC_MODEL_HUGE

__SDCC_USE_XSTACK
when —xstack option is used
__SDCC_STACK_TENBIT

__SDCC_MODEL_FLAT24

__SDCC_VERSION_MAJOR
Always defined. SDCC major version number. E.g. 3 for SDCC 3.5.0
__SDCC_VERSION_MINOR

__SDCC_VERSION_PATCH
