3.5.3 SM83 intrinsic named address spaces


3.5.3.1 __sfr

The keyword __sfr is an alternative way to access memory locations 0xff00 to 0xffff, which are typically used for memory-mapped I/O e.g.:

__sfr __at(0xff01) IoPort;
